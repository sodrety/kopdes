# Preserve existing Loan obligations when adding Loan Types

Accepted. Existing pending Loan Requests are removed when Loan Types are introduced, together with their partial approval history and actionable Officer notifications. Each affected Member receives a notification to submit a replacement request with a Loan Type. Existing approved, rejected, and cancelled Loan Requests and all existing Loans are classified as **Regular — Legacy Terms**; existing fees, installment schedules, and outstanding balances are preserved without recalculation and displayed using their actual preserved Admin Fee. Type-based Admin Fee rules apply only to Loan Requests submitted after this change. This avoids retroactively changing Member obligations while ensuring no untyped pending request enters the new approval policy.
